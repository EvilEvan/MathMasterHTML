## üéØ Project Description

MathMaster HTML is an engaging educational game designed to help students master algebraic concepts through progressive difficulty levels. The application combines mathematical problem-solving with visual feedback, creating an environment where learning algebra becomes both challenging and enjoyable.

Students progress through three distinct levels, each focusing on different mathematical operations, while being guided by an intuitive lock-based progression system that provides immediate feedback and encouragement.

## üöÄ Getting Started

### Prerequisites

- A modern web browser (Chrome 60+, Firefox 55+, Safari 12+, Edge 79+)
- Basic understanding of algebra concepts (recommended but not required)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/EvilEvan/MathMasterHTML.git
   cd MathMasterHTML
   ```

2. **Navigate to the application directory**
   ```bash
   cd MathMaster-Algebra
   ```

3. **Open the application**
   - **Option A**: Double-click `index.html` to open in your default browser
   - **Option B**: Serve with a local web server (recommended)
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     
     # Using PHP
     php -S localhost:8000
     ```

4. **Access the application**
   - If using a local server, navigate to `http://localhost:8000`
   - If opening directly, the file will open in your browser

## üìñ Usage Instructions

### Starting Your Journey

1. **Welcome Screen**: Click anywhere or press Enter/Space to continue from the animated welcome screen
2. **Level Selection**: Choose your difficulty level:
   - **Beginner** (üå±): Focus on addition and subtraction
   - **Warrior** (üî•): Multi-operation challenges
   - **Master** (‚ö°): Advanced division problems

3. **Game Interface**: The game features a three-panel layout:
   - **Left Panel**: Problem display and lock progression
   - **Center Panel**: Problem-solving workspace with help system
   - **Right Panel**: Matrix-style symbol rain for ambiance

### Navigation Controls

- **Mouse**: Click buttons and interactive elements
- **Keyboard Shortcuts**:
  - `1`, `2`, `3`: Quick level selection
  - `Escape` or `Backspace`: Go back to previous screen
  - `Enter` or `Space`: Continue/Submit actions
- **Touch**: Full touch support for mobile devices

### Problem Solving

1. View the algebraic equation in the left panel
2. Use the workspace in the center panel to solve step-by-step
3. Input your answer to progress through the lock mechanism
4. Receive immediate feedback and advance to the next problem
